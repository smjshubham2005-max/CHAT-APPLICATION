# Real-Time Chat Application

This project is a simple real-time chat application developed as part of an internship task.

## Technologies Used
- Node.js
- Express.js
- Socket.IO
- HTML, CSS, JavaScript

## Description
The application allows multiple users to communicate with each other in real time using WebSocket technology. Messages sent by one user are instantly received by all connected users.

## How to Run
1. Install dependencies using `npm install`
2. Start the server using `node server.js`
3. Open browser and go to `http://localhost:3000`
